# Movies
Training app - the Movie DB client. Movies list with pagination + description screen. 
- MVP
- Moxy
- Retrofit
- Navigation Component + safe args
- RxJava3
- Dagger
- Glide 
- instrumented tests: Mockk, Espresso, FragmentScenario
